[h1][b]Start with all Research[/b][/h1]
[b]Fastest track for all the smart duplicants...[/b]

[h1]Description[/h1]
Adds a toggle to the meteor creation screen which, when enabled, will give the colony all research nodes. Made for the purpose of testing, however, can be used for a casual play through.

[h2]Mod Version: [b]1.0.2[/b][/h2]
[h2]Latest Tested Game Version:[b] CS-444111[/b][/h2]

[h1]Specifics[/h1]
[list]
    [*] Adds a toggle button to the New Game creation screen.
    [*] When enabled will force the research tree to be completely learnt.
[/list]
