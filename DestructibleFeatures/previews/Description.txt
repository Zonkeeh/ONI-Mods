[h1][b]Destructible Features[/b][/h1]
[b]Provides a balanced means to remove geysers/volcanoes.[/b]

[h1]Description[/h1]
Had the issue where an unwanted feature is in the way of your builds?

This mod aims to resolve this by providing a balanced and configurable solution to remove these features.

[h2]Mod Version: [b]1.0.6[/b][/h2]
[h2]Latest Tested Game Version:[b] CS-444111[/b][/h2]

[h1]Specifics[/h1]
[list]
[*] Duplicants require the superduperhard digging skill to remove a feature.
[*] Feature has to of been studied to have the option of destruction.
[*] Balances the Study + Destruction work time to give easy access for removal.
[*] Effects all existing features, no need to start a new colony.
[*] Duplicants will gain xp for this interaction, however, no resources are dropped upon completion (Possible future update).
[*] Speed of destruction also scales with a duplicant's excavation skill.
[/list]

[h1]Configuration[/h1]
[i]Config.json file can be found in:
[list]
[*] Windows: ~\Documents\Klei\OxygenNotIncluded\mods\Steam\1869090131\
[*] Mac:  ~/Library/Application Support/unity.Klei.Oxygen Not Included/mods/Steam/1869090131/
[/list]

A template file Config.json.template is provided, simply copy, remove the .template from the filename and remove the comments (//).[/i]

Config has the following variables:
[list]
    [*] RemoveNeutronium - [i]Option will remove the neutronium base upon destruction.[/i] (default: true)
    [*] ReplaceNeutroniumWithObsidian - [i]Will replace the neutronium with obsidian (false will cause the neutronium to be destoryed into a vaccum).[/i] (default: true)
    [*] AnaylsisTime - [i]Integer time between 1 and 10000 for how long (ingame seconds) a study task should take.[/i] (default:2400)
    [*] DeconstructTime - [i]Integer time between 1 and 10000 for how long (ingame seconds) a destruction task should take.[/i] (default:800)
