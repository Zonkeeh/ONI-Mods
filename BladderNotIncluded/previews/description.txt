[h1][b]Bladder Not Included[/b][/h1]
[b]Duplicants no longer require a functioning Bladder![/b]


[h1]Description[/h1]
Ever wanted to stop duplicants from wetting themselves, or complaining whilst you test out your latest build?

This mod enables a world setting option to stop duplicants from using their bladder. This option can also be forced to all existing saves.



[h2]Mod Version: [b]1.0.1[/b][/h2]
[h2]Latest Tested Game Version:[b]CS-444111[/b][/h2]



[h1]Specifics[/h1]
[list]
    [*] Bladders are automatically cleared.
    [*] Duplicants will also no gain any negative bladder effects.
    [*] Option can be found in game settings on new meteor creation.
[/list]

[h1]Options[/h1]
Configuration of this mod and a basic template can be found in the following locations:
[list]
[*][b]Windows[/b] •  ~\Documents\Klei\OxygenNotIncluded\mods\Steam\2044107742\Config.json.template
[*][b]Mac[/b] •  ~/Library/Application Support/unity.Klei.Oxygen Not Included/mods/Steam/2044107742/Config.json.template
[/list]

[b]ForceLoad[/b] - Allows the mod to treat all saves as 'Bladder Not Included', meaning worlds generated before this option can still be included. [i](Default: false)[/i]

                                                                                          [i][b]Disclaimer:[/b] In the case where invalid arguments are given the default values will be used.[/i]
